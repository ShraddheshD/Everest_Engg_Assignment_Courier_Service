# Everest Engineering Assignment

This repository contains an assignment solution and automated tests for Everest Engineering. It targets .NET 10 and includes unit tests for core components such as the Offer Service, Shipment Planner, and Cost Calculator.

## Contents

- `Assignment_Everest_Engineering` — main application project.
- `Assignment_Test_Cases` — unit test project containing tests for key services (`OfferServiceTests`, `ShipmentPlannerTests`, `CostCalculatorTests`).

## Prerequisites

- .NET 10 SDK
- Visual Studio 2026 (recommended) or the `dotnet` CLI
